# Quantity Measurement Application

This project is a backend application developed using Java 17 and Spring Boot. The application is used to perform different quantity measurement operations such as unit conversion, comparison, addition, subtraction, multiplication, and division for different measurement types like Length, Weight, Volume, and Temperature. The project also stores operation history in the database for future reference. 

The application is built using Spring Web for creating REST APIs, Spring Data JPA and Hibernate for database operations, and MySQL/PostgreSQL for data storage. Lombok is used to reduce boilerplate code and Maven is used for dependency management and project build. APIs are tested using Postman.

The project supports:

* Unit Conversion
* Quantity Comparison
* Arithmetic Operations
* Database Storage of History
* Error Handling for Invalid Operations

Technologies Used:

* Java 17+
* Spring Boot
* Spring Web
* Spring Data JPA
* Hibernate
* MySQL / PostgreSQL
* Maven
* Lombok
* Postman

This project helped in understanding REST API development, database integration, service layer implementation, exception handling, and unit conversion logic using Spring Boot.
